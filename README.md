# LawPilot
LawPilot membantu siapa pun memahami dokumen legal tanpa latar belakang hukum. Pengguna mengunggah foto atau PDF kontrak, perjanjian, atau dokumen resmi. Gemini merangkum isi dokumen dalam bahasa sederhana, menandai klausul berisiko, dan menyarankan pertanyaan penting untuk diajukan kepada pengacara atau notaris sebelum menandatangani.
